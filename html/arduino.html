Arduino


Arduino é uma plataforma de hardware e software de código aberto que possibilita o desenvolvimento de projetos eletrônicos. Ele permite que qualquer pessoa crie, monte e modifique seu próprio arduino. O arduino é composto por um microcontrolador Atmel, circuitos, entradas e saídas.



Adapatado de: Tinkercad

Entrada digital


A entrada digital é uma porta que recebe sinal digital por um sistema ou dispositivo que são reconhecido em dois estados de tensão: "Ligado" e "Desligado".


A entrada pode ser reconhecida em 3 tipos; Volts, Fisico e Binário.

Volts: Ligado = 5V ; Desligado = geralmente perto dos 0V.

Binário: Ligado = 1 ; Desligado = 0.

Físico: Ligado = HIGH ; Desligado = LOW.


Saida Analógica


A saida analógica é o processo em que o microcontrolador converte um valor digital em um sinal analógico, tensão ou corrente para controlar dispositivos. Por mais que a saída analógica não se aplica ao arduino (pois não possui opção de saída analógica), ela também é composta por duas respostas: "Ligado" e "Desligado".

Volts: ligado = 5V ; Desligado = geralmente perto dos 0V.

Binário: Ligado = 1 : Desligado = 0.

Físico: Ligado = HIGH; Desligado = LOW.


microcontrolador


Um microcontrolador basicamente é usado para criar projetos interativos e eletrônicos. Ele também é responsável por executar programas e ver as saídas e entradas.

O modelo de um microcontrolador do Arduino Uno (utilizado em sala de aula) é o ATmega328P.

Já o fabricante desse modelo do microcontrolador é o prórprio ATMEL(Atual microchip).

Multímetro


Multímetro é uma forma de medir as grandezas elétricas, geralmente utilizado para ver o funcionamento das ferramentas.

Um multímetro mede as grandezas chamadas:

V (Volts) - Tensão

A (Amperes) - Corrente (contínua e alternada)

Ω (Ohms) - Resistência


Lei de Ohm


A lei que o multímetro aplica para realizar as medidas é chamada "Lei de Ohm". A lei de Ohm nos permite calcular grandezas físicas, como a tensão, corrente e resistência.


Fonte: mundodaeletrica

V = Tensão elétrica (V)

I = Corrente elétrica (A)

R = Resistência elétrica (Ω)


Teste de continuiade


O teste de continuidade serve para saber se há um rompimento ou defeito no componente.

No multímetro, você utiliza a função de "continuidade" que tem a imagem de uma onda sonora. Liga a ponta preta no COM, e a ponta vermelha ligada no V/mA/Ω, e toca cada ponta nas extremidades do componente. Se caso ocorrer um barulho, significa que está funcionando, que tem continuidade e que o circuito está fechado.


Medição do Resistor


Para fazer a medição de uma resistência no resistor, utilizamos o multímetro na função ligada em ohms(Ω) e encostamos as pontas em cada lado do resistor, mostrando o número de ohms que o resistor tem.

Se caso você não sabe a quantidade de ohms que o resistor que você vai medir possui, gire a chave para selecionar o número mais alto de ohms, e vai descendo conforme o valor vai se aproximando no visor. É importante fazer a medição com o resistor desconectado do circúito para evitar problemas.


Medir Tensão no Círcuito Com Resistor e LED


Para medir a tensão no circuito com resistor e LED, utilizaremos o multímetro com a função em tensão contínua em 20V, com as pontas preta(COM) e vermelha(V/mA/Ω) no multímetro.

Medir a tensão no led: A ponta vermelha é conectada no lado positivo do LED e a ponta preta é conectada na parte negativa do LED. Para medir a tensão do LED.



Medir a tensão no resistor: A ponta vermelha é conectada no lado negativo do LED e a ponta preta é conectada no resistor que vai para o negativo.



Medição de Uma Corrente Elétrica no Circuito


Para medir a corrente elétrica no circuito, a gente liga as pontas preta(COM) e vermelha(V/mA/Ω), e ligamos o multímetro na corrente contínua(A), geralmente no valor mais alto (10A), mas diminuindo conforme o tempo.

No circuito, iremos fazer a troca do fio que conecta o resistor e led, utilizando as pontas do multímetro no lugar.

Conectamos a ponta vermelha ao lado do resistor, onde a corrente entra.

Conectamos a ponta preta ao lado do led, onde a corrente sai.

O valor que aparece no multímetro é o valor da corrente que está passando pelo ponto do circuito.

Se caso você inverter as pontas, o valor sai negativo.

Sistema de Medição Analógico Com LEDs


O sistema de medição analógico com leds consiste nos niveis de iluminamentos da entrada analógica que, dependendo da quantidade, liga um led específico.

Aqui está o protótipo do Sistema no Tinkercad.



Aqui está a parte física do sistema, junto com um vídeo que demonstra a funcionamento.




Para saber quantos volts tem quando a medida da tensão chega no valor de 300, é só fazer uma regra de 3. Onde: X = 5 * 300 / 1023

O valor obtido foi de: 1.5V



O código do Sistema de medição analógico que é responsável pelo funcionamento é:


// C++ code
//
int nivel = 0;

void setup()
{
  pinMode(A0, INPUT);
  Serial.begin(9600);
  pinMode(11, OUTPUT);
  pinMode(12, OUTPUT);
  pinMode(13, OUTPUT);

  nivel = analogRead(A0);
}

void loop()
{
  nivel = analogRead(A0);
  Serial.println(nivel);
  if (nivel >= 300) {
    digitalWrite(11, HIGH);
  } else {
    digitalWrite(11, LOW);
  }
  if (nivel >= 600) {
    digitalWrite(11, LOW);
    digitalWrite(12, HIGH);
  } else {
    digitalWrite(12, LOW);
  }
  if (nivel >= 900) {
    digitalWrite(12, LOW);
    digitalWrite(13, HIGH);
  } else {
    digitalWrite(13, LOW);
  }
  delay(10); // Delay a little bit to improve simulation performance
}
